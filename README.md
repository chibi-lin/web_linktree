# 個人作品集網站

## 📖 專案簡介

這是一個專為藝術家設計的個人作品集網站。

## 技術相關

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **字體**: Google Fonts (Karla, Kanit)
- **圖示**: Material Symbols, Font Awesome
- **部署**: GitHub Pages
- **網域管理**: Cloudflare
- **響應式**: CSS Grid, Flexbox
